# blog-website

Minimalist blog website created in Javascript

*Front End Technologies*
* HTML5
* CSS3
* Bootstrap

*Back End Technologies*

* Node.js
* Express

## *Installation*

### 1. Clone the repo:
`git clone https://github.com/Carlos2MorenoM/blog-website.git`

### 2. Install NPM packages:
`npm install`

### 3. Run
`node app.js`
